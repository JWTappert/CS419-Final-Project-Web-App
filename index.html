CS419 Final Project: Heap Mapping

This is a web application for our final project in CS419: Internet of Things. We have decided to create a heat map of a residence using a wireless sensor network. We are using a Raspberry Pi 3, three Arduino Uno boards with heat sensors attached as well as Xbee WiFi shields. The RP3 is acting as the base station for the sensors and queries the data that is stored in a MySQL database. The base station code is written in Python, the code on each of the nodes is C++, and finally the web application is written in PHP. Our team consists of Adam Duquette and Justin Tappert.
